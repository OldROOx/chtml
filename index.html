BLOG TÉCNICO — AWS
▶ TUTORIAL 01 · AWS EC2
DESPLEGAR UNA APP NODE.JS EN AWS EC2
Cómo lanzar una instancia EC2 Ubuntu, instalar Node.js y dejar corriendo una API REST accesible desde internet.
  • Crear instancia EC2 t2.micro (free tier)
  • Conectarse vía SSH con archivo .pem
  • sudo apt install nodejs npm
  • node server.js & (background)
  • Abrir puerto 3000 en Security Group
▶ TUTORIAL 02 · NGINX
NGINX COMO PROXY INVERSO EN EC2
Configura Nginx para redirigir tráfico del puerto 80 a tu app Node.js en 3000, ocultando el puerto y preparando la ruta para HTTPS.
  • sudo apt install nginx
  • Editar /etc/nginx/sites-available/default
  • proxy_pass http://localhost:3000
  • sudo systemctl restart nginx
  • Verificar con curl http://tu-ip-publica
▶ TUTORIAL 03 · LOAD BALANCER
BALANCEO DE CARGA CON NGINX EN AWS
Configura Nginx como balanceador entre múltiples instancias EC2 para alta disponibilidad y tolerancia a fallos.
  • Levantar 2+ instancias EC2 con la misma app
  • Configurar bloque upstream en Nginx
  • upstream backend { server IP1; server IP2; }
  • proxy_pass http://backend
  • Probar caída de un nodo sin perder servicio
